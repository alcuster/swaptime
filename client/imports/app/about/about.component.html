How to use

  • 1. Submit a listing by clicking "Submit" in the menu on the left. You can either request a course or offer one of your own.
  • 2. Browse other users' listings on the "All Listings" page, and if something catches your eye, click on the listing. On the next page, you can start a chat thread with that user.
  • 3. Using the in-app messaging system, coordinate your trade. The safest way to conduct a course swap is for both parties to visit the Registrar's office.
  • 4. When your trade is complete, please delete the listing (or, if it belongs to another user, request that he/she do so).

Info

This web app was built with Angular 2 and MeteorJS (united via angular-meteor), and uses angular/material2 and PrimeNG UI components.
